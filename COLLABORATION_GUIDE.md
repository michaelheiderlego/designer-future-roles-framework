## ✨ Super Simple Collaboration

**Everyone's changes are saved in real-time - just enter your name!**

### How to Use (30 seconds):

1. **Open the site**
2. **Enter your name** when prompted
3. **Start editing** - that's it!

Your changes appear instantly for everyone. No accounts, no tokens, no complexity!

---

## 🎯 What You'll See

### First Visit:
- Popup: **"Welcome! 👋 Please enter your name:"**
- Type your name (e.g., "Sarah")
- Click OK

### While Editing:
- Top banner shows: **"Editing as [Your Name]"**
- See who else is active: **"Also editing: Michael, Anna"**
- Changes save automatically
- **No refresh needed** - see updates in real-time!

---

## 🚀 How to Use

### Adding Items

1. **Circular View**: Click any colored circle → Click **"+ Add"**
2. **List View**: Toggle to List View → Click **"+ Add"** in any zone

### For AI-Augmented Zone:
- Add **Links** to tools/resources
- Add **Screenshot URLs** to show how-tos
  - Upload screenshots to [imgur.com](https://imgur.com)
  - Copy image URL
  - Paste in "Screenshot URL" field

### Editing & Deleting

- **Edit**: Click ✏️ icon
- **Delete**: Click 🗑️ icon
- Changes sync instantly across all devices

### Change Your Name

- Click **"Change Name"** button (top right)
- Enter new name
- Others will see updates under your new name

## 🎨 Best Practices

### DO:
- ✅ Use clear, descriptive titles
- ✅ Add context in descriptions
- ✅ Test links before adding
- ✅ Build on colleagues' contributions
- ✅ Ask before deleting others' items

### DON'T:
- ❌ Delete items without discussing
- ❌ Use vague titles like "Thing 1"
- ❌ Add duplicate content
- ❌ Edit while someone is presenting (wait for breaks)

## 🎯 Tips for Great Contributions

### Human Core Zone:
- Focus on irreplaceable human judgment
- What can AI never do?
- What should designers always control?
- Be specific about the skill

### Human Checkpoints Zone:
- Critical decision gates
- When does a human need to review?
- What are the quality standards?
- Define clear criteria

### AI-Augmented Zone:
- **Specific tools**: Name the tool
- **How to use**: Brief instructions
- **Links**: Direct link to tool
- **Screenshots**: Show the workflow
- **Prompts**: Share what works

**Example AI tip:**
```
Title: Figma AI for Layout Variations
Description: Generate 5 layout options in seconds. Select frames,
right-click → "Generate variants". Review and refine the best one.
Link: https://figma.com/ai
Screenshot: [imgur link to workflow]
```

## 🆘 Troubleshooting

### "Welcome" popup doesn't appear
- You might have dismissed it - click **"Change Name"** button

### My changes aren't saving
- Check internet connection
- Look for error message in banner
- Try refreshing the page

### I don't see others' changes
- Hard refresh: Cmd/Ctrl + Shift + R
- Verify you're all on the same site
- Check if they actually saved (ask them)

### Name is wrong
- Click **"Change Name"** button (top right)
- Enter correct name
- Previous edits keep your old name

### Can't edit/buttons not working
- Refresh the page
- Clear browser cache
- Try different browser

## 🎉 Workshop Activities

### Icebreaker (5 min):
- Everyone adds one human skill they value most
- Watch the core zone fill up
- Discuss overlaps and surprises

### Deep Dive (30 min):
- Assign zones to small groups
- Each group adds 3-5 items
- Present back to full team

### AI Tools Share (20 min):
- Everyone adds one AI tool they use
- Include link and how-to
- Demo the most interesting ones

### Future Skills (15 min):
- What skills should we develop?
- Add to Human Core
- Vote on priorities (discuss)
